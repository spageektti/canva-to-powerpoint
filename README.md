# canva-to-powerpoint

Convert Canva presentations to PowerPoint.

## Instructions
- Download the Canva presentation as GIFs.
- Check if everything looks fine.
- Run the `main.py` in the directory with GIFs files.
- Your presentation is saved to `output.pptx`.
